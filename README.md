# League of Legends Analytics Dashboard

![Dashboard](assets/dashboard-demo.gif.gif)


## Sobre o projeto

Este projeto consiste em um dashboard interativo desenvolvido em Power BI para análise de desempenho de partidas de League of Legends.

A proposta foi transformar dados extraídos da Riot Games API em uma experiência visual inspirada na identidade do jogo, combinando análise de dados, storytelling e design de interface.

## Objetivos

* Analisar desempenho do jogador
* Explorar estatísticas de campeões
* Identificar padrões de vitória e derrota
* Apresentar insights de forma visual e intuitiva
* Criar uma experiência inspirada no universo de League of Legends

## Tecnologias utilizadas

* Power BI
* Python
* Riot Games API
* Excel
* Figma

## Principais indicadores

* Win Rate
* KDA Médio
* Campeões mais jogados
* Lane favorita
* Melhor horário para jogar
* Melhor dia da semana
* Histórico de partidas
* Dano causado
* Ouro acumulado


## Coleta de dados

Os dados foram obtidos através da Riot Games API, tratados utilizando Python e posteriormente modelados para análise no Power BI.
